# 远程文件管理器

Android 远程文件管理器，支持 WebDAV 和 SMB3 两种协议。

## 功能特性

- 🌐 **WebDAV 协议** - 连接支持 WebDAV 的文件服务器
- 📂 **SMB3 协议** - 连接 Windows SMB 共享，支持内网穿透
- 📁 **文件浏览** - 浏览、导航远程目录
- ⬆️ **文件上传** - 从本地上传文件到远程服务器
- ⬇️ **文件下载** - 下载远程文件到本地（可自由选择保存位置）或直接打开
- 📝 **新建文件夹** - 在远程服务器创建新目录
- ✏️ **重命名** - 重命名文件和文件夹
- 🗑️ **删除** - 删除远程文件或文件夹
- 🔄 **连接管理** - 保存和管理多个服务器连接
- ✏️ **编辑连接** - 修改已保存的连接信息
