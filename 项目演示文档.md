# Android 远程文件管理器 - 项目演示文档

## 一、项目介绍

本项目是一款 Android 远程文件管理器应用，支持通过 WebDAV 和 SMB3 协议连接远程服务器，实现远程文件的浏览、管理操作。应用采用现代 Android 开发技术栈构建，界面美观、操作流畅。

### 技术栈
- **语言**: Kotlin
- **UI 框架**: Jetpack Compose + Material 3
- **网络库**: OkHttp (WebDAV)、jcifs-ng (SMB3)
- **架构**: MVVM (ViewModel + StateFlow)
- **目标平台**: Android 16 (API 36)
- **最低支持**: Android 8.0 (API 26)

---

## 二、功能清单

| 功能 | 描述 |
|------|------|
| 远程连接管理 | 添加、编辑、删除 WebDAV/SMB3 服务器连接配置 |
| WebDAV 协议支持 | 支持标准 WebDAV 协议的 PROPFIND/GET/PUT/MKCOL/DELETE/MOVE 操作 |
| SMB3 协议支持 | 支持 SMB3 共享文件浏览和管理（内网穿透） |
| 文件列表展示 | 列表形式显示文件/文件夹，含名称、大小、修改时间、类型图标 |
| 目录导航 | 进入子目录、返回上级、返回根目录、路径栏快速跳转 |
| **文件上传** | 从本地选择文件，上传至远程服务器，支持大文件缓冲流传输 |
| **文件下载** | 支持两种模式：①下载到本地（自由选择保存位置）②直接打开预览 |
| 新建文件夹 | 在当前目录下创建新文件夹 |
| 重命名 | 对文件/文件夹进行重命名操作 |
| 删除 | 删除远程文件或文件夹 |
| 连接状态持久化 | 保存连接配置，应用重启后仍可使用 |
| 使用说明 | 内置中文使用说明文档 |

## 五、功能操作步骤

### 5.1 添加 WebDAV 连接
1. 打开应用，点击右上角 **+** 按钮
2. 选择 **WebDAV** 协议标签
3. 填写连接名称（如"我的NAS"）
4. 填写服务器地址（如 `http://192.168.1.100`）
5. 填写用户名和密码
6. 点击 **保存**

### 5.2 添加 SMB3 连接
1. 点击右上角 **+** 按钮
2. 选择 **SMB3** 协议标签
3. 填写连接名称
4. 填写服务器 IP 地址（如 `192.168.1.100`）
5. 填写共享名（如 `shared`）
6. 填写用户名和密码
7. 点击 **保存**

### 5.3 浏览远程文件
1. 在连接列表中点击 **连接** 按钮
2. 连接成功后自动显示根目录文件列表
3. 点击文件夹进入子目录
4. 点击顶部路径栏中的节点可快速跳转
5. 点击左上角返回箭头回到连接列表

### 5.4 文件操作
- **新建文件夹**: 点击右上角文件夹+图标，输入名称后确认
- **重命名**: 长按文件/文件夹，在弹出菜单中选择"重命名"
- **删除**: 长按文件/文件夹，在弹出菜单中选择"删除"
- **刷新**: 点击右上角刷新按钮重新加载当前目录
- **上传文件**: 点击右下角上传按钮，选择本地文件上传到当前目录
- **下载到本地**: 长按文件 → 选择"下载到本地" → 选择保存位置（Downloads/文档等）
- **打开文件**: 长按文件 → 选择"打开" → 先下载到缓存再调用系统应用打开

### 5.5 查看使用说明
- 在连接列表页面点击右上角 **?** 图标
- 在文件列表页面点击右上角 **?** 图标

## 六、界面说明

### 6.1 连接列表页面
- 顶部标题栏：显示"远程文件管理器"，右侧有帮助和添加按钮
- 连接卡片：显示服务器名称、协议类型、地址、用户名
- 每个卡片有"连接"和"删除"按钮
- 空状态时显示提示文字和云朵图标

### 6.2 文件列表页面
- 顶部标题栏：显示连接名称，有返回、帮助、新建文件夹、刷新按钮
- 右下角操作区：上传按钮（主按钮）和下载按钮（次要按钮）
- 路径导航栏：显示当前路径，可点击节点快速跳转
- 文件列表：每个文件显示类型图标、名称、大小、修改时间
- 文件夹显示文件夹图标，右侧有箭头指示
- 空文件夹显示空状态提示
- 长按任意文件/文件夹：弹出操作菜单（下载到本地/打开/重命名/删除）

## 七、编译打包方法

### 7.1 环境准备
1. 安装 JDK 17 或更高版本
2. 安装 Android Studio 或 Android SDK
3. 配置 `ANDROID_HOME` 环境变量

### 7.2 编译步骤
```bash
# 进入项目目录
cd FileManagerApp

# 编译 Debug APK
./gradlew assembleDebug

# 编译 Release APK
./gradlew assembleRelease
```

### 7.3 输出位置
- Debug APK: `app/build/outputs/apk/debug/app-debug.apk`
- Release APK: `app/build/outputs/apk/release/app-release-unsigned.apk`

### 7.4 安装到设备
```bash
# 通过 ADB 安装
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 八、桌面服务器部署说明

### 8.1 WebDAV 服务器 (端口 80)
使用项目中的 `启动WebDAV服务器.bat` 脚本：
1. 确保已安装 Python 3.x
2. 双击运行脚本
3. 脚本会自动安装 wsgidav 并启动服务
4. 默认用户名: `admin`，密码: `admin123`

### 8.2 SMB3 共享 (端口 445)
使用项目中的 `配置SMB3共享.bat` 脚本：
1. 以管理员身份运行脚本
2. 脚本会创建 `C:\SMBShare` 共享文件夹
3. 使用 Windows 用户名密码连接
4. 如需专用用户：`net user smbuser password123 /add`
